7. Кроме простых property необходимо так же задать ссылки на другие классы. Какие есть способы это сделать? - Чтобы
задавать ссылки на другие классы необходимо использовать атрибут ref=""; Пример:
<property name="country" ref="country"/> - ссылаемся на бин c id country

8. Один из способов связать бины между собой - указать явно сссылку на нужный бин <property name=.. ref=...>. Каковы
достоинства и недостатки этого подхода? При таком подходе нам не надо заново компилировать.
Недостатки: можно ошибится в написание и много кода писать.

9. Проблема может найти нето + заново компилировать. Плюсы: лаконичность
